    What is HTML? HTML Stands for Hyper Text Markup Language.Its standard markup laguage used to create web pages and display content on the internet.
    HTML is the backbone of a website,providing the stucture and content that the web browser renders to the user.Its made up of a series of elements,represented by TAGs(<>),which wrap around content to define its purpose and display properties.

    HTML is used to:

  1. Define headings, paragraphs,and other text elements
  2. Create links to other web pages or email addresses
  3. Add images, videos, and audio files
  4. Build tables, lists, and other data structures
  5. Create forms for user input
  6. Apply basic styling and layout control

    7. HTML is a crucial skill for web development, and is often used in conjuction with other technologies like CSS (Cascading Style Sheet) and JavaSript to create dynamic and interactive web pages.

    Some common HTML tags include:

    • <h> Heading h1-h6
    • <p> Paragraph
    • <a> Anchor (link)
    • <img> Image
    • <br> Line break
    • <hr> Horizontal Rule
    Here's a quick summary of each element:
    1. - Headings (h1-h6): Define the title or heading of a section or page
    2. - Paragraphs (p): Define a paragraph of text
    3. - Links (a): Create a hyperlink to another web page or email address
    4. - Images (img): Add an image to a web page
    5. - Line Breaks (br): Create a line break in a paragraph of text
    6. - Horizontal Rule (hr): Create a horizontal line to separate content

    HTML is an essential building block of the web, and is used by web browsers to render web pages and display content to users.

For Class & Id

In HTML, class and id are two attributes used to identify and style HTML elements.

  • Class:

    - The class attribute assigns a class name to an element.
    - Multiple elements can share the same class name.
    - Classes are used to apply styles, layouts, or behaviors to elements.
    - You can add multiple classes to an element, separated by spaces.
    Example:
    <p class="text-center bold">This text is centered and bold.</p>

  • ID:
    - The id attribute assigns a unique identifier to an element.
    - Each element can only have one ID.
    - IDs are used to target specific elements with CSS or JavaScript.
    - IDs must be unique within the HTML document.
    Example:
    <p id="main-title">Main Title</p>

For Checkboxes

In HTML, a checkbox is a type of input element that allows users to select one or more options from a list. The syntax is:

  • <input type="checkbox" id="checkboxID" name="checkboxName" value="checkboxValue">

    - id is a unique identifier for the checkbox
    - name is the name of the checkbox group
    - value is the value associated with the checkbox You can also add additional attributes, such as:
    - checked to make the checkbox selected by default
    - disabled to make the checkbox non-selectable
    - onclick to specify a JavaScript function to call when the checkbox is clicked

For Iframe tag

The iframe tag is used to embed another HTML document or a web page inside a current HTML document.
The syntax is:

  • <iframe src="[URL]" [attributes]>
    - src: specifies the URL of the page to embed
    - attributes: additional attributes to customize the iframe (e.g., width, height, border, scrolling)

Example:

This code embeds the webpage at the specified URL in an iframe with a width of 500 pixels, a height of 300 pixels, no border, and scrolling enabled.
Some common attributes for iframes include:
- src: specifies the URL of the page to embed
- width and height: specify the size of the iframe
- frameborder: specifies the border style (0 for no border)
- scrolling: specifies whether to enable scrolling (yes or no)
- marginwidth and marginheight: specify the margin sizes
- align: specifies the alignment (left, right, top, middle, bottom)

Note: iframes can be used for good (e.g., embedding maps or videos) or evil (e.g., clickjacking or phishing). Always use them responsibly!
